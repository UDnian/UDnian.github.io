过滤选择器


















































code type demo
基本过滤选择器
:first 单个 $(”div:first”)选取所有<div>元素中的第一个<div>元素
:last 单个 $(”div:last”)选取所有<div>元素中的最后一个<div>元素
:not(selector)集合$(”input:not(.myClass)”)选取class不是myClass的<input>元素
:even 集合$(”input:even”)选取索引是偶数的input元素
:odd 集合$(”input:odd”)选取索引是奇数的input元素
:eq(index) 单个 $(”input:eq(1)”)选取索引等于1的input元素
:gt(index)集合$(”input:gt(1)”)选取索引大于1的input元素
:lt(index)集合$(”input:lt(1)”)选取索引小于1的input元素
:header集合$(”:header”)选取所有<h1>,<h2>…标题元素
:animated 集合$(”div:animated”)选取正在执行动画的div元素
内容过滤选择器
:contains(text)集合$(”div:contains(’我’)”)文本中有’我’的div
:empty集合$(”div:empty”)不含子元素(包括文本)的div元素
:has(selector)集合$(”div:has(p)”)含有P元素的div元素
:parent集合$(”div:parent”)有子元素(包括文本)的div元素
可见性过滤选择器
:hidden集合$(”:hidden”)所有不可见元素,包括<input type=”hidden”/>,<div style=”display:none;”>,<div style=”visibility:hidden;”>等元素。如果只想选取<input>元素,可以用$(”input:hidden”)
:visible 集合$(”div:visible”)所有可见的<div>元素
属性过滤选择器
[attribute]集合$(”div[id]”)拥有属性id的元素
[attribute=value] 集合$(”div[title=test]”)属性title为”test”的div
[attribute!=value]集合$(”div[title!=test]”)属性title不等于”test”的<div>元素(包括没有title的div)
[attribute^=value]集合属性title以”test”开始的div
[attribute$=value]集合属性title以”test”结束的div
[attribute*=value]集合属性title含有”test”的div
[selector1] [selector2] [selectorN]$(”div[id][title$=test]”)拥有属性id,并且属性title以test结束的div
子元素过滤选择器
:nth-child(index/even/odd/equation) 集合:eq(index)只匹配一个元素,而:nth-child将为每一个父元素匹配子元素,从1开始。而:eq(index)是从0算起的
:nth-child(3n)集合能选取每个父元素下的索引值是3的倍数的元素(n从0开始)
:first-child 集合$(”ul li:first-child”);选取每个<ul>中的第一个li元素
:last-child集合$(”ul li:last-child”);选取每个<ul>中的最后一个li元素
:only-child集合$(”ul li:only-child”);在ul中选取是惟一子元素的li元素
表单对象属性过滤选择器
:enabled 集合$(”#form1:enabled”)id为form1内的可用元素
:disabled 集合$(”#form2:disabled”)不可用元素
:checked 集合$(”input:checked”)所有被选中的input
:selected 集合$(”select:selected”)被选中的选项元素

jQuery中的DOM操作

DOM Core

js中的getElementById()、getElementsByTagName()、getAttribute()、setAttribute()等方法，都是DOM Core组成部分
获取表单对象：
document.getElementsByTagName(”form”);
获取某元素的src属性的方法：
element.getAttribute(”src”);

HTML_DOM

使用HTML_DOM获取表单对象的方法：
document.forms  //表单数组
使用HTML_DOM获取某元素的src属性的方法：
element.src;

CSS_DOM

CSS_DOM针对CSS的操作。
设置某元素style对象字体颜色的方法:
element.style.color = ”red”;

code type demo
append() 向每个匹配的元素内部追加内容 <p>hello!</p>$(”p”).append(”<b>world</b>”);
<p>hello!<b>world</b></p>
appendTo() 颠倒append()方法 <p>hello!</p>$(”p”).appendTo(”<b>world</b>”)
;<p><b>world</b>hello!</p>
prepend() 向每个匹配的元素内部前置内容 <p>hello!</p>$(”p”).prepend(”<b>world</b>”)
;<p><b>world</b>hello!</p>
prependTo() 颠倒prepend()方法 <p>hello!</p>$(”<b>world</b>”).appendTo(”p”);
<p><b>world</b>hello!</p>
after() 在每个匹配的元素之后插入内容 <p>hello!</p>$(”p”).after(”<b>world</b>”);
<p>hello!</p><b>world</b>
insertAfter() 颠倒after()方法 <p>hello!</p>$(”p”).insertAfter(”<b>world</b>”);
<b>world</b><p>hello!</p>
